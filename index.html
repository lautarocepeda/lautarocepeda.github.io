CANJE DE BENEFICIOS/CUPONES

  • Esta aplicación se encarga de gestionar beneficios/descuentos de un comercio y que los clientes puedan hacer uso de ellos.

    Este proyecto surgió cuando me propuse desarrollar una aplicación que me aportara nuevos conocimientos, implementar tecnologías que no había usado e ir aprendiendo en el transcurso que la iba desarrollando.

    Quería una aplicación donde el cliente estuviera totalmente separado del servidor (cliente/servidor) y que usara tokens para la autenticación. Asi que hice un REST API con Slim Framework (PHP) para consumir datos con el Front, que lo hice en Angular. Para la autenticación use "JWTokens".
